X TKJ 1-Presentasi

Web yang kami buat bersama dalam rangka presentasi mengenai sedikit materi yang diajarkan pada X TKJ 1. Kabel UTP, Topology Jaringan, Data center dan Cloud Computing, Subnet Mask. Web ini mengajarkan kami dalam membuat slide show yang menarik dengan bantuan HTML, CSS, JavaScript. Namun dalam project pertama ini ada beberapa fitur yang tidak dapat digunakan ataupun visual yang kurang menarik. Dikarenakan presentasinya telah selasai maka kami tidak akan membuat update/perubahan pada web ini, tapi dengan bug yang ada kami bisa belajar untuk memperbaiki di project kedepannya.
